# What am I building ?

I am building a small open source llm. I have named it Atropus. Atropus's main task is being a story generation brain. It will be trained on books corpus and many more narratives to make it able to generate narratives easily.


## Why am I building it ?

### Mimicking Human Brain

Human thinking is not a clean, linear pipeline. It is messy, iterative, and pattern-driven.

When faced with a situation, the brain does not evaluate every possible variable explicitly. Instead, it:

Recognizes patterns from past experiences
Generates multiple possible thoughts or interpretations
Selects the one that historically led to better outcomes

This project is an attempt to model that process.

From Story Generator → Thought Engine

Atropus starts as a story generation model, but that is only the surface layer.

The deeper goal is to evolve it into a thought generation system:

Given a situation, it produces multiple plausible “thought paths”
Each path represents a different interpretation or action trajectory
These generated thoughts can then be evaluated by a separate decision-making model
